[https://colab.research.google.com/drive/12BWaZ3uXNvrCy5-J5jGHxPEW5TCADGab?usp=sharing] - Competição de Heróis / Vacinação (Aula 19): Introdução à modularização do código através da criação de funções customizadas (def), escopo de variáveis e retorno de resultados (return).
